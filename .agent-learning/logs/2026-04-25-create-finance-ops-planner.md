# Finance Ops Planner Agent Creation

## Observation

The user needs a finance/operations expert to estimate costs and unit economics for automated content generation, autoposting, and early growth of PsyCalc / Cognitive Matchmaker.

## Action

Created a dedicated agent specification:

- `finance-ops-planner`

## Safeguards

- The agent is not a licensed financial, tax, legal, accounting, securities, or investment advisor.
- It must frame budgets as assumptions/scenarios, not guarantees.
- It must avoid guaranteed ROI claims and route experiment analysis to data/statistics agents when needed.

## Typical use

- Estimate monthly content automation budget.
- Calculate cost per generated/approved/posted video.
- Model CAC and funnel assumptions.
- Compare cheap / normal / aggressive growth scenarios.
