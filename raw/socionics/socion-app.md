# Socion App - Socionics Dating/Matching Platform

Source: socion.app and socionicsinsight.com

## Overview

Socion is a dating/matching app built on Socionics theory. It tests intertype relations at scale with empirical data.

## Key Features

1. **You choose the dynamic, not a black-box algorithm**
   - Select which intertype relations you're open to
   - Dual for deep complementarity
   - Mirror for intellectual sparring
   - Transparent matching logic

2. **Confidence scoring, not binary labels**
   - Uses probability distribution across likely types
   - Refines over time based on behavioral signals
   - Handles typing uncertainty honestly

3. **Post-match feedback**
   - Generates empirical data on relation type outcomes
   - Tests: Do Dual pairs report higher satisfaction?
   - Tests: Which relations correlate with longer conversations?
   - Tests: Does typing accuracy predict match outcome?
