# Socionics Subtypes (DCNH)

DCNH subtypes describe four possible behavior patterns for each socionic type. Unlike the socionic type itself, subtypes can change under prolonged environmental influence.

## The Four Subtypes

| Subtype | Description | Latin Name |
|---------|-------------|------------|
| **D** | Dominant | Dominant |
| **C** | Creative | Creative |
| **N** | Normalizing | Normalizing |
| **H** | Harmonizing | Harmonizing |

## Subtype Characteristics

### Dominant (D)
- Emphasizes the 1st function
- Seeks to realize the leading function
- May appear aggressive in their area of strength

### Creative (C)
- Emphasizes the 2nd function
- Creates new in their area
- More adaptive but less stable

### Normalizing (N)
- Emphasizes the 3rd function
- Follows social norms
- Balances between creative and harmonizing

### Harmonizing (H)
- Emphasizes the 4th function
- Seeks harmony and balance
- May appear weak but provides stability
