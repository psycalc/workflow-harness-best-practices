# Intertype Relations Mapper Scoring

Source: intertype-relations-calculator.neocities.org

## Ranking System

Rankings from 1 (best) to 16 (worst):

| Rank | Relation |
|------|----------|
| 1 | Dual (Soulmate) |
| 2 | Identical (Same type) |
| 3 | Activity (Chill friends) |
| 4 | Mirror (Introverted/Extraverted versions) |
| 5 | Kindred (Share 1st function) |
| 6 | Look-a-like (Share 2nd function) |
| 7 | Semi-dual (Same as soulmate but different 2nd function) |
| 8 | Illusionary (Same as soulmate but different 1st function) |
| 9 | Super-ego (Each type most detrimental to society) |
| 10 | Benefit: First selected type is Benefactor |
| 11 | Benefit: First selected type is Beneficiary |
| 12 | Supervision: First selected type is Supervisor |
| 13 | Supervision: First selected type is Supervisee |
| 14 | Quasi-identical (Rival) |
| 15 | Contrary (Ultimate Rival) |
| 16 | Conflict |

## Scoring Calculation

- **Group average score**: Averages ranks of all relationships drawn
- **Individual average score**: Averages ranks of all lines connected to a particular type

**Note:** Lower score = better compatibility (opposite of typical interpretation)

## Features

- Visual spatial mapping of intertype relations
- Color-coded lines by relation type
- Calculate averages for groups or individuals
