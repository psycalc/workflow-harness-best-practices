# Reinin Traits in Socionics

Reinin traits are additional dichotomies beyond the basic Jungian ones. They help distinguish between types within the same MBTI category.

## Core Reinin Traits

| Trait | Description |
|-------|-------------|
| **Static/Dynamic** | Fixed patterns vs. changing behavior |
| **Positivism/Negativism** | Focus on positive vs. negative aspects |
| **Constructivism/Emotivism** | Logical vs. emotional approach |
| **Tactics/Strategy** | Short-term vs. long-term orientation |
| **Carelessness/Providence** | Spontaneous vs. planning |
| **Yielding/Stubborn** | Compliant vs. resistant |
| **Questim/Declarative** | Questioning vs. declarative communication |
| **Aristocracy/Democracy** | Status-based vs. equality-based |

## Function Properties (7 Properties)

Functions are defined by combinations of seven properties:

| Property Pair | Description |
|---------------|-------------|
| **Mental/Vital** | Consciousness vs. subconscious processing |
| **Acceptance/Production** | Receptive vs. generative processing |
| **Strong/Weak** | Ability under pressure |
| **Valued/Unvalued** | Importance to the person |
| **Inert/Contact** | Engagement style |
| **Loaded/Unloaded** | Energy expenditure |
| **Template/Situational** | Fixed vs. flexible approach |

## Dimensionality Groups

Functions cluster into groups based on properties:

- **Mental functions**: 1st, 2nd, 3rd, 4th (conscious block)
- **Vital functions**: 5th, 6th, 7th, 8th (unconscious block)
