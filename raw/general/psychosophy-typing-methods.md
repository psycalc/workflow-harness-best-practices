## 2. Методология типирования

### 2.1 Три дихотомии (достаточно 2 для определения типа)

| Дихотомия | Описание | Позиции |
|-----------|----------|--------|
| **ВЕРХ/НИЗ** | Высокая (1-2) / Низкая (3-4) функция | 1,2 = верх / 3,4 = низ |
| **Процессионность/Результативность** | Процесс (диалог) / Результат (монолог) | 1,4 = результат / 2,3 = процесс |
| **Доминирование/Подстройка** | Самостоятельность / Гибкость | 1,3 = доминирование / 2,4 = подстройка |

### Формулы функций

```
1-я функция: результативная + доминирующая (высокая)
2-я функция: процессионная + подстраивающаяся (высокая)
3-я функция: процессионная + доминирующая (низкая)
4-я функция: результативная + подстраивающаяся (низкая)
```

### 2.2 Матричный метод (6 вопросов)

Каждый вопрос — выбор между двумя аспектами. Формируется матрица 4×4:

**Вопросы:**
1. Бывает ли, что вы пренебрегаете едой или сном, если решили закончить работу к сроку? (Воля-Физика)
2. Случается ли, что под влиянием эмоций вы совершаете поступки, о которых жалеете? (Эмоция-Воля)
3. Может ли ваше самочувствие измениться под влиянием настроения? (Эмоция-Физика)
4. Часто ли вы сознательно совершаете поступки, которые кажутся вам нелогичными? (Воля-Логика)
5. Бывает ли так, что вы действуете в ущерб материальной выгоде, следуя убеждениям? (Логика-Физика)
6. Можете ли вы сознательно сказать глупость ради развлечения окружающих? (Эмоция-Логика)

**Алгоритм:**
- Инициализировать матрицу 4×4 нулями
- Для каждого вопроса: "+" → строка аспекта +1, "-" → столбец аспекта +1
- Сортировка по убыванию: наибольшее = 1-я позиция, наименьшее = 4-я

### 2.3 Метод перечисления приоритетов

Самый быстрый метод (11 из 14 точных результатов в тесте).

**Инструкция для типируемого:**
"Весь мир можно разделить на 4 аспекта:
- Физика — тело, материальное: еда, внешность, собственность, комфорт
- Эмоция — душа: чувства, переживания, интуиция, творчество
- Логика — разум: мышление, анализ, знания, мнения
- Воля — дух: цели, желания, уверенность, самоконтроль

Расположите эти 4 аспекта по порядку важности для вас — от самого важного до наименее важного."

**Примеры типов:**
- В.И. Ленин: Воля → Логика → Физика → Эмоция = ВЛФЭ
- Л.Н. Толстой: Воля → Эмоция → Физика → Логика = ВЭФЛ

## 3. Акцентуации (8 на каждую функцию)

### Система свойств

Каждая функция характеризуется 3 свойствами:

| Свойство | Варианты |
|---------|----------|
| Высокая/Низкая | 1,2 = высокая / 3,4 = низкая |
| Процессионная/Результативная | 1,4 = результат / 2,3 = процесс |
| Доминирование/Подстройка | 1,3 = доминирование / 2,4 = подстройка |

### Акцентуации по позициям

**Для 1-й функции (высокая, результативная, доминирующая):**
- Акцентуация неосознанности (результативность)
- Акцентуация интровертности (принципиальность)
- Акцентуация субъектности (высокая)

**Для 2-й функции (высокая, процессионная, подстраивающаяся):**
- Акцентуация осознанности (процессионность)
- Акцентуация экстравертности (непринципиальность)
- Акцентуация субъектности (высокая)

**Для 3-й функции (низкая, процессионная, доминирующая):**
- Акцентуация осознанности
- Акцентуация интровертности
- Акцентуация объектности

**Для 4-й функции (низкая, результативная, подстраивающаяся):**
- Акцентуация неосознанности
- Акцентуация экстравертности
- Акцентуация объектности

## 4. Определение акцентуаций

### Вопросы для определения (примерные)

**Для 3-й Воли:**
1. Когда принимаете важное решение — долго анализируете или ищете поддержку?
2. Как реагируете на критику "у вас нет воли"?
3. В стрессовой ситуации — берёте себя в руки сами или ищете "отмашку"?

### Методы определения акцентуаций

1. **Наблюдение за поведением** — какое свойство проявляется ярче всего
2. **Тестирование** — специализированные вопросы
3. **Интервью** — вопросы о поведении в различных ситуациях

1. **Проходить тест несколько раз** — в разное время суток и эмоциональном состоянии
2. **Комбинировать методы** — тест + интервью + самотипирование
