## Как использовать для типологий

### Вариант 1: Быстрое исследование (30 минут)

```
Задача AI:
"Найди все исследования о связи соционических типов 
и удовлетворённости в отношениях. 
Сделай summary с методологией и результатами."
```

**Инструменты:**
- GPT Researcher (бесплатно, open source)
- Perplexity AI (бесплатно)
- You.com

### Вариант 2: Полное исследование (1-2 дня)

```
Задача AI:
"Проведи исследование: дуальные пары (соционика) 
счастливее чем конфликтные?

1. Найди существующие исследования
2. Сформулируй гипотезу
3. Предложи дизайн эксперимента
4. Напиши код для анализа данных
5. Сгенерируй отчёт"
```

**Инструменты:**
- Agent Laboratory (open source)
- The AI Scientist
- Custom GPT + Code Interpreter

### Вариант 3: Автономное исследование (день-два)

**The AI Scientist может:**
- Самостоятельно придумать исследование
- Написать код
- Запустить эксперименты
- Написать статью
- Провести self-review

**Но:** требует настройки, мощности, данных

## Конкретные инструменты

### 1. GPT Researcher ⭐ (Бесплатно, Open Source)

```bash
# Установка
pip install gpt-researcher

# Использование
from gpt_researcher import GPTResearcher

query = "socionics intertype relations couples satisfaction research"
researcher = GPTResearcher(query=query)
report = await researcher.conduct_research()
```

**Что делает:**
- Ищет 20+ источников
- Суммирует с цитированиями
- Генерирует отчёт 2000+ слов
- **Время: 2-3 минуты**
- **Цена: ~$0.10**

### 2. Perplexity AI (Бесплатно)

Просто задай вопрос:
```
"Какие исследования подтверждают что дуальные пары 
в соционике счастливее? Дай ссылки."
```

Ответит с источниками за секунды.

### 3. ChatGPT/GPT-4 с Code Interpreter

```
"У меня есть CSV с данными 100 пар:
- тип мужчины (1-16)
- тип женщины (1-16)
- счастье (1-10)

Проверь гипотезу: дуальные пары счастливее.
Напиши код, запусти, покажи результаты."
```

### 4. Claude with Analysis Tool

```
"Проанализируй прикреплённый датасет.
Проверь корреляцию между типом отношений
и удовлетворённостью."
```

### 5. The AI Scientist (Продвинутым)

**GitHub:** github.com/SakanaAI/AI-Scientist

**Требует:**
- GPU (для запуска экспериментов)
- Настройка
- 4-8 часов работы

## Практический сценарий для типологий

### День 1: Литературный обзор (AI)

**Утро:**
```
GPT Researcher: "Обзор исследований соционики и отношений"
→ 10-страничный отчёт за 5 минут
```

**Обед:**
```
Perplexity: "Что известно о валидности MBTI?"
→ Краткий summary
```

### День 2: Генерация гипотез (AI)

```
ChatGPT-4: 
"На основе этих исследований сгенерируй 5 
проверяемых гипотез о связи типов и счастья в парах"
```

### День 3: Дизайн и сбор данных

```
ChatGPT:
"Напиши анкету в Google Forms для проверки гипотезы.
Включи вопросы о типах и шкалу счастья."
```

### День 4: Анализ (AI)

```
Code Interpreter:
"Вот данные 87 пар. Проведи анализ:
1. t-тест дуальные vs конфликтные
2. Корреляции
3. Визуализация"
```

### День 5: Отчёт (AI)

```
Claude:
"Напиши научный отчёт по результатам.
Включи: введение, методы, результаты, обсуждение."
```
