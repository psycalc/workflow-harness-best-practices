# Typology Authors Timeline

> **Status:** Research in progress — This document catalogs who developed which typological system and when. Critical for understanding intellectual lineage and avoiding attribution errors.

---

## Chronological Overview

```
1921       Jung — Psychological Types
1940s      Myers & Briggs — MBTI (drawing on Jung)
1940s-90s  Eysenck — PEN model (separate trait tradition)
1960s-90s  Lexical/factor tradition — Big Five / Five-Factor Model
1970s      Augustinavičiūtė — Socionics (drawing on Jung + Kępiński)
1993       Afanasyev — Psychosophy / Psyche-Yoga
late 1990s Temporistics — Nika Sherman / Dr. Radut + Alexander Latyshev
```

## Individual Authors

### Carl Gustav Jung (1875-1961)

| Attribute | Value |
|-----------|-------|
| **Nationality** | Swiss |
| **Field** | Analytical Psychology |
| **Years Active** | 1900s-1961 |
| **Key Work** | *Psychological Types* (1921) |

**Contributions:**
- Introduced 4 psychological functions: Thinking, Feeling, Sensing, Intuition
- Introduced attitude types: Extraversion and Introversion
- Defined the "psychic energy" (libido) concept
- Established a major foundation for later Jungian and type-based systems, especially MBTI and related traditions

**Key Publication:**
```
Jung, C.G. (1921). Psychologische Typen. Zürich: Rascher.
[English: Psychological Types, 1971]
```

**Notes:**
- Jung did NOT use MBTI dichotomies (J/P, S/N in current form)
- His system was qualitative, not quantitative
- Extraversion/Introversion = direction of psychic energy

---

### Hans Jürgen Eysenck (1916-1997)

| Attribute | Value |
|-----------|-------|
| **Nationality** | German-born, UK |
| **Field** | personality psychology |
| **Years Active** | 1940s-1990s |
| **Key Work** | PEN Model |

**Contributions:**
- Developed the PEN model: Psychoticism, Extraversion, Neuroticism
- Emphasized biological/biochemical basis for personality
- Introduced cortical arousal theory (introverts have higher baseline arousal)
- Used factor analysis for empirical research

**Key Publication:**
```
Eysenck, H.J. (1967). The Biological Basis of Personality. Springfield, IL: Thomas.
Eysenck, S.B.G. & Eysenck, H.J. (1976). Psychoticism and Impulse Control. In J. N. Butcher (Ed.), 
Research in Personality (Vol. 2). London: Pergamon.
```

**Notes:**
- "Psychoticism" in his model = antisocial behavior, NOT psychosis
- Eysenck's Extraversion = cortical arousal level, NOT the same as Jung's
- This is a common source of confusion — same word, different construct

---

### Aušra Augustinavičiūtė (1927-2005)

| Attribute | Value |
|-----------|-------|
| **Nationality** | Lithuanian |
| **Field** | Socionics / Information Metabolism Theory |
| **Years Active** | 1970s-2005 |
| **Key Work** | Model A theory |

**Contributions:**
- Founded Socionics (Соционика), extending Jung's work
- Developed Model A — structure of 8 cognitive functions
- Introduced 16 information metabolism types
- Systematized intertype relationships (Supervision, Mirage, Dual, etc.)

**Key Publication:**
```
Augustinavičiūtė, A. (1990). Model and Theory in Structure of Information Metabolism. 
Vilnius: LTOU. [in Lithuanian]
```

**Notes:**
- Based her work on Jung's psychological types PLUS Antoni Kępiński's information metabolism theory
- NOT the same as MBTI — Socionics has different function structure
- MBTI and Socionics types do NOT map 1:1 (e.g., ENTJ ≠ LIE despite appearances)

**Other Names:**
- Sometimes spelled "Augustinaviciute" or "Augustinavichyute"
- Some secondary sources interpret "Model A" as referring to Augustinavičiūtė, but this attribution should be treated cautiously

---

### Antoni Kępiński (1918-1972)

| Attribute | Value |
|-----------|-------|
| **Nationality** | Polish |
| **Field** | Psychiatry / Neuroscience |
| **Years Active** | 1950s-1972 |
| **Key Work** | Information Metabolism Theory |

**Contributions:**
- Developed the theory of information metabolism (metabolizm informacyjny)
- Proposed that psyche processes information like an organism processes substances
- Proposed information metabolism as a model of organism-environment informational exchange
- Augustinavičiūtė later drew on his information-metabolism concept when formulating Socionics

**Key Publication:**
```
Kępiński, A. (1972). Introduction to the Study of Information Metabolism. 
Warsaw: PZWL. [in Polish]
```

**Notes:**
- Died in 1972, before Socionics emerged as a distinct typology
- His work is lesser-known in English due to language barriers
- Important for understanding the information-metabolism metaphor later used in Socionics

---

### Alexander Afanasyev (1940s-?)

| Attribute | Value |
|-----------|-------|
| **Nationality** | Russian |
| **Field** | Personality Typology / Psychosophy |
| **Years Active** | 1990s-present |
| **Key Work** | "Syntax of Love" (Синтаксис любви), 1993 |

**Contributions:**
- Created Psychosophy (also known as Psychoyoga or Attitudinal Psyche)
- Model: 4 functions (Will, Emotion, Logic, Physics) × 4 positions
- 24 base types + theoretical 81 comprehensive types
- Based partially on Jung and Berdyaev

**Key Publication:**
```
Afanasyev, A. (1993). Syntax of Love (Синтаксис любви). Moscow. [in Russian]
```

**Notes:**
- 25th "type" = 2-2-2-2 — person with balanced development
- Different from Socionics — uses different function set
- "Function" in Psychosophy = different from Jungian "function"

## Attribution Guidance

### Common Errors to Avoid

| Error | Correction |
|-------|------------|
| "Jung invented MBTI" | MBTI was created by Myers & Briggs, based on Jung |
| "Eysenck invented Big Five" | Eysenck developed PEN; the Big Five emerged from the lexical/factor-analytic tradition and was later operationalized prominently by Costa & McCrae |
| "Socionics = MBTI" | Different systems with different function models |
| "Psychosophy = Socionics" | Different function sets; different theoretical basis |
| "Extraversion means the same thing everywhere" | Different in Jung, Eysenck, Big Five, Socionics |

### Correct Attribution

```
Jung → Psychological Types (1921)
  → Myers/Briggs → MBTI (1940s)
  → Augustinavičiūtė → Socionics (1970s), alongside Kępiński's information-metabolism metaphor

Lexical / factor-analytic tradition → Big Five / Five-Factor Model
Eysenck → PEN model

Afanasyev → Psychosophy (1993)
Nika Sherman / Dr. Radut + Alexander Latyshev → Temporistics (late 1990s), drawing on Afanasyev and Berdyaev among other influences
```

## Notes on This Project

We use **Socionics** as the primary tactical-level system because:
1. It has a more elaborated internal literature than Psychosophy or Temporistics
2. It offers a clearer information-metabolism metaphor for this project's modeling needs
3. It has a more developed intertype relationship tradition
4. Its function-based model allows for more nuanced compatibility analysis

All three niche typologies remain weakly supported in mainstream personality science, so this project's use of them should be treated as exploratory rather than settled validation.
