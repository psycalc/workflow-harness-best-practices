# Socionics Reinin Dichotomies

## Overview

Reinin dichotomies are an additional layer of type splits beyond the four Jungian dichotomies. Their usefulness and even validity are disputed across Socionics schools.

## Classical / Wikisocion Position

Wikisocion treats Reinin dichotomies as a recognized classical Socionics layer:

- 15 dichotomies total
- 4 Jungian foundation pairs
- 11 derived non-Jungian pairs

Wikisocion also explicitly notes that the usefulness of many or most Reinin dichotomies is questioned by many socionists.

For `SLI`, Wikisocion lists:

- judicious
- objectivist
- aristocratic
- result
- carefree
- obstinate
- dynamic
- strategic
- constructivist
- positivist
- asking

## Imperative Socionics Position

Imperative Socionics challenges part of the classical Reinin set.

In the source snippets currently available to the project, it treats at least the following as **false traits**:

- aristocracy / democracy
- process / result
- asking / declaring
- positivism / negativism

This means these traits should not be treated as universally valid across schools.

## Practical Project Rule

- Use the Jungian foundation with the highest confidence.
- Use classical Reinin traits only with explicit confidence labels.
- Mark school-disputed traits as `school-dependent` or `contested`.
- Do not let Reinin traits override clearer Model A evidence.

## Recommended Confidence Split

### High confidence

- introvert / extravert
- irrational / rational
- logic / ethics
- sensing / intuition

### Medium confidence

- traits directly listed in a classical table for a type and not strongly contradicted in current use

### Contested

- traits explicitly challenged by another school in the current source set
