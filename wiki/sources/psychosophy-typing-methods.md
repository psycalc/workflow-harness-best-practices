## Методология типирования

### Три дихотомии

1. **ВЕРХ/НИЗ** — высокие (1-2) / низкие (3-4) функции
2. **Процессионность/Результативность** — процесс / результат
3. **Доминирование/Подстройка** — самостоятельность / гибкость

### Матод перечисления приоритетов

Самый быстрый метод (11 из 14 точных результатов):
1. Спросить: "Расположите 4 аспекта по порядку важности"
2. Физика → Эмоция → Логика → Воля
3. Результат: порядок определяет тип
