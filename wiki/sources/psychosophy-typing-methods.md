# Методы типирования по психософии

Обзор методов, тестов и лучших практик для определения психософского типа и акцентуаций.

## Методология типирования

### Три дихотомии

1. **ВЕРХ/НИЗ** — высокие (1-2) / низкие (3-4) функции
2. **Процессионность/Результативность** — процесс / результат
3. **Доминирование/Подстройка** — самостоятельность / гибкость

### Метод перечисления приоритетов

Быстрый screening-метод, пригодный только для первичной гипотезы, а не для окончательного типирования:
1. Спросить: "Расположите 4 аспекта по порядку важности"
2. Физика → Эмоция → Логика → Воля
3. Результат: порядок может подсказать рабочую гипотезу о типе

Числовые claims вроде “11 из 14 точных результатов” следует считать anecdotal/community-reported, пока не указаны критерий истинного типа, blinded procedure, sample size, inter-rater reliability и независимая репликация.

## Акцентуации

Акцентуации — unofficial/community extension, а не core Afanasyev doctrine. В текущих источниках их следует описывать как дополнительные property-accent patterns; конкретное число и структура вариантов требуют отдельной source verification.

### Определение акцентуаций

1. Наблюдение за поведением
2. Специализированные вопросы
3. Интервью

## Рекомендации

- Проходить тест несколько раз
- Комбинировать методы
- Внешняя проверка у нескольких типировщиков с явным уровнем уверенности
- Не трактовать тесты как психометрически валидированные инструменты без reliability/validity данных
