# Intertype Relations Matrix

## Overview

Reference page for Socionics intertype relation classes. This page is **not** an empirically calibrated compatibility-score table: relation labels describe theory-internal tactical information exchange, not guaranteed relationship outcomes.

## Quick Reference — Relation Classes, Not Scores

| Relation class | Tactical mechanism summary | Caveat |
|----------------|----------------------------|--------|
| Identity | Same Model A structure; easy recognition, shared blind spots | Same type does not mean total compatibility |
| Duality | Reciprocal valued-weak complement (`1↔5`, `2↔6`) | Not a soulmate guarantee |
| Activation | Mutual valued-function stimulation | Energizing can also become overstimulating |
| Mirror | Shared values with leading/creative inversion | Similar themes, different emphasis |
| Benefit / Request / Social Order | Asymmetric suggestive-function activation | Direction matters: benefactor/beneficiary roles differ |
| Supervision / Revision / Audit | Asymmetric vulnerable-function pressure | Direction matters: supervisor/supervisee roles differ |
| Business / Cooperation | Strong-function coordination without deep weak-zone relief | Often task-readable, not necessarily intimate |
| Mirage / Illusionary | Partial comfort with unstable alignment | Initial ease does not imply stable fit |
| Quasi-identity | Similar themes with valuation mismatch | Looks similar at surface, differs structurally |
| Extinguishment / Contrary | Reciprocal damping of information initiative | Not necessarily hostile |
| Conflict | Reciprocal strong-to-weak valuation clash | High friction hypothesis, not inevitable conflict |
| Super-Ego | Mutual super-ego strain/formality | Can remain civil at distance |
| Kindred / Comparative | Shared leading orientation with value divergence | Recognition without full complement |
| Semi-duality | Incomplete complementary coverage | Partial duality is not calibrated as “almost duality” |

## Scoring Status

No percentages are currently assigned here because PsyCalc has not calibrated Socionics relation classes against observed dyadic outcomes. Any future score must specify:

- outcome being predicted;
- sample and typing reliability;
- uncertainty intervals;
- baseline comparison;
- calibration method.
